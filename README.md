# Binary Bloom Vault

A collaborative post depository built using Python.

## Features

- User Authentication
- Password Hashing
- Public Post Feed
- Binary Version Tracking
- Branch Suggestions
- History Logs
- JSON Data Storage

## Technologies Used

- Python
- JSON
- hashlib
- dataclasses
